

# Library Inventory Manager

A command-line application for managing a library's book inventory. Tracks book status, allows searching, and persists data to a text file.

## Features
- Add, issue, and return books
- Search by title or ISBN
- View all books
- Persistent storage with error handling
- Logging for operations

## Usage
Run the CLI: `python cli/main.py`

## Requirements
- Python 3.x
- Dependencies: logging (built-in)
